# Obsidian Configuration & Scripts

This repository contains various configuration files, scripts, and snippets for [Obsidian](https://obsidian.md/).

## Structure

- **DataViewJS/**: A collection of [DataviewJS](https://blacksmithgu.github.io/obsidian-dataview/api/intro/) scripts for advanced data querying and visualization.
- **CSS/**: Custom CSS snippets to modify the look and feel of Obsidian.

## Future Plans

More categories and resources will be added as the collection grows.
